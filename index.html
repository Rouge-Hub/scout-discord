Command deck

What Scout can do

/help

Command guide

Shows every available command with a short description.

/server

Server info

Displays server details, owner, boosts, channels, and features.

/user

User lookup

Shows account, avatar, roles, and server join information.

/remind

Temporary reminders

Sets a reminder while Scout is running.

/roll

Dice roller

Rolls formats like 1d20, 2d6, or 4d10.

@Scout

Optional AI chat

When configured, Scout can answer mentions and image questions.
